## Step 1: Open This Link
👉 **https://share.streamlit.io**

## Step 2: Click "New app"

## Step 3: Fill These Fields
```
Repository: abdmananch16/awwaz360

```

## Step 4: Click "Deploy"
⏳ Wait 2-3 minutes...

## Step 5: Add Your API Keys
After app loads:
1. Click ⚙️ (gear icon) → Settings
2. Click "Secrets"
3. Paste this:
```
TAVILY_API_KEY = "your-tavily-key-here"
HF_API_KEY = "your-huggingface-key-here"
```

## Step 6: Get Free API Keys (2 minutes)

### Tavily (Free)
1. Go: https://tavily.com
2. Sign up
3. Copy API key
4. Paste in Secrets

### Hugging Face (Free)
1. Go: https://huggingface.co/settings/tokens
2. Create new token
3. Copy token
4. Paste in Secrets
